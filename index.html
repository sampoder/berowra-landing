Frequently Asked Questions

What is Deta Space?
Berowra is built to run on Deta, a serverless cloud hosting provided. Deta Space is an abstraction of this cloud hosting which allows individuals to install private apps it their own personal cloud. Read more here.
What types of data are supported?
Berowra supports short strings, numbers, longer strings with markdown, dates, string arrays, files and colours at the moment.
What is the storage limit of Berowra?
Deta Space limits each app to 10GB of files stored, this is therefore Berowra's limit.
Where can I provided feedback, request features, or report issues?
Please use our GitHub issues.
